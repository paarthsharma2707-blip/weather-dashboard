# 🌤️ Weather Dashboard

A modern Weather Dashboard built with HTML, CSS, and JavaScript that displays live weather information using the Open-Meteo API.

## ✨ Features

- 🌍 Search weather by city
- 🌡️ Real-time temperature
- 💨 Wind speed & direction
- 💧 Humidity
- 🌧️ Precipitation
- 🔭 Visibility
- 📊 Atmospheric pressure
- ☀️ UV Index with risk indicator
- 📅 7-Day Weather Forecast
- 📱 Fully Responsive Design
- ⚡ Smooth animations
- 🌙 Modern Dark UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo Weather API
- Open-Meteo Geocoding API
